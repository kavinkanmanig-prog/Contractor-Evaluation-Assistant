# Contractor-Evaluation-Assistant# Contractor Evaluation Assistant

## Overview

Contractor Evaluation Assistant is an AI-powered chatbot developed for Chennai Metro Rail Limited (CMRL) to automate contractor performance assessment. The system analyzes contractor-related documents and performance data, evaluates key performance indicators (KPIs), and provides intelligent scoring and insights to support data-driven decision-making.

## Features

* AI-powered conversational interface
* Retrieval-Augmented Generation (RAG) based document analysis
* Contractor performance scoring
* KPI evaluation and ranking
* PDF document processing
* Automated insight generation

## Technologies Used

* Python
* Streamlit
* Generative AI
* Natural Language Processing (NLP)
* Retrieval-Augmented Generation (RAG)
* PDF Processing Libraries

## Project Structure

* `app.py` – Main chatbot application
* `cmrl_rag.zip` – RAG implementation files
* `CMRL AI Sample.pdf` – Sample project document
* `CMRL_Tech_Usage.txt` – Technology usage documentation

## Applications

This solution can be used by government organizations, infrastructure companies, and project management teams to evaluate contractor performance efficiently and consistently.
